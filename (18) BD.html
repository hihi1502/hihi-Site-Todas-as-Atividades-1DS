Banco de Dados

Fizemos um banco de dados lógico de uma livraria. O Banco de Dados lógico é um tipo de modelo de dados que descreve elementos de dados em detalhes e é usado para desenvolver entendimentos visuais de entidades de dados, atributos, chaves e relacionamentos. É a ultima etapa para um banco de dados físico.
